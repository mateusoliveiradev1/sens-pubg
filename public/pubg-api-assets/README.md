# PUBG API weapon assets

These weapon images are copied from the official `pubg/api-assets` GitHub repository for PUBG API developers:

https://github.com/pubg/api-assets

Source folder:

`Assets/Item/Weapon/Main/`

The upstream README says the repository provides art assets and data dictionaries for use with the PUBG API and that use is governed by PUBG developer Terms of Use and Player-created Content rules. These assets must stay non-exclusive and must not be presented as official endorsement, sponsorship, or a paid-only PUBG data feature.

`JS9` is not present in the upstream `api-assets` repository at the time this asset set was added, so the UI keeps the authored fallback silhouette for that one weapon.
