<threat_model>
Scope: premium feature enforcement and visible free/pro projection across product surfaces.

Threats:
- HIGH: premium data is hidden only in React while server actions still return full data. Mitigation: server-side projection helpers and read actions enforce access before payload reaches components.
- HIGH: another user's history/trends/outcomes leak through client-supplied ids. Mitigation: every read/write remains scoped to `auth().user.id` and tests cover cross-user denial.
- MEDIUM: Free becomes a broken demo and loses trust. Mitigation: projection keeps truth evidence, verdict, mastery, confidence/coverage/inconclusive state, diagnosis summary, and short next step.
- MEDIUM: locks deceive with fake/blurred data. Mitigation: locked surfaces use honest previews and explicit blocker reasons.
- MEDIUM: community Pro ideas accidentally paywall public community. Mitigation: tests prove public feed/posts/basic engagement remain free/open where already open.
</threat_model>
