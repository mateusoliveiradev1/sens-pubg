# Research: Pitfalls

**Date:** 2026-05-05

## Pitfall 1: Selling Certainty Before Measurement Earns It

Warning signs:

- Copy says "perfect sens" or "guaranteed improvement."
- Coach recommends hard values on low-confidence clips.
- Benchmarks do not cover the promised use case.

Prevention:

- Define precision and perfection as measurable targets.
- Show confidence, coverage, and "inconclusive" states.
- Gate aggressive recommendations behind multi-clip evidence.

Phase to address: Phase 1 through Phase 4.

## Pitfall 2: Serving Three Customer Types With One Message

Warning signs:

- Landing/pricing copy talks to casuals, competitive grinders, and teams all at once.
- Product decisions mix solo training, community, and B2B coach workflows in the same release.

Prevention:

- Launch the first paid wedge for solo competitive/dedicated players.
- Treat teams/coaches as expansion after the solo workflow proves retention.

Phase to address: Phase 5 and Phase 6.

## Pitfall 3: Paywalling PUBG API-Derived Data

Warning signs:

- Premium tier unlocks official PUBG API stats or official match data unavailable to free users.
- The product name/branding implies official PUBG endorsement.

Prevention:

- Monetize original clip analysis, coach plans, history, and training workflow.
- Keep API data, if added, as contextual support.
- Include clear non-affiliation language.

Phase to address: Phase 5.

## Pitfall 4: Free Tier Costs More Than It Converts

Warning signs:

- Unlimited free analyses.
- LLM calls for non-paying users.
- Large storage or support burden before conversion signals exist.

Prevention:

- Keep free analysis useful but limited.
- Run LLM only when paid, rate-limited, or explicitly controlled.
- Track activation, conversion, retention, and provider cost per user.

Phase to address: Phase 5 and Phase 7.

## Pitfall 5: Coach Sounds Better But Does Not Improve Outcomes

Warning signs:

- More polished text without outcome tracking.
- No way for player to mark protocol accepted, completed, improved, or failed.

Prevention:

- Add coach outcome tracking.
- Benchmark tier, primary focus, next-block protocol, and repeat-clip improvement.
- Use memory to reduce repeated bad advice.

Phase to address: Phase 4.
