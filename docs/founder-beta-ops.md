# Founder Beta Operations

Phase 5 is a controlled solo-player beta, not public launch.

## Cohorts

| Cohort | Size | Purpose |
|---|---:|---|
| Internal sanity | 3-5 | Catch broken flows before interpreting value. |
| Beginner/casual | 5-10 | Check clarity, capture guidance, and Free usefulness. |
| Serious/competitive | 5-15 | Check repeat-clip value, coach usefulness, and upgrade pressure. |
| Coach/creator/advisor | 1-3 | Qualitative review with material-benefit disclosure. |

## Creator Rules

- Grant, discount, access, or invite code requires disclosure acceptance first.
- Phase 5 supports attribution tags only.
- No affiliate payout, referral commission, or creator revenue share exists in Phase 5.

## Rollout Gates

- Grant-first preview can run with manual admin grants.
- Founder paid allowlist opens only after automated gates, manual Stripe test-mode checklist, and explicit owner approval.
- Public launch waits for later validation, premium UI, complete training, revenue operations, community Pro, and team phases.

## Support Rules

- Cancellation and payment failure never delete history.
- Refund/dispute/fraud cases follow Stripe truth plus audited support notes.
- Admin can grant/revoke/suspend/reconcile. Support can view and add notes only.
- Pricing and billing copy must remain independent from PUBG/KRAFTON and avoid perfect sensitivity, guaranteed rank, or guaranteed improvement claims.
